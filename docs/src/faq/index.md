# FAQ

## Do I have to use Docker?

Yes, that's how this project is packaged.

This makes it easier to support the project when we have control over the version of DNSMasq other packages
use by the project.

## I can't get my DNS to work properly?

Your best bet is to create an issue on [GitHub](https://github.com/Zephyris-Pro/dnsmasq-manager/issues).

## My language is not available, how can I contribute?

If your language is not currently supported, you’re very welcome to help us add it!

To contribute, please follow the documentation available on the wiki on [GitHub](https://github.com/Zephyris-Pro/dnsmasq-manager). The wiki explains the full process step by step, including:

* How to create or request a new language file
* The required format and structure
* Contribution guidelines and best practices
* How to submit your changes via pull request

Make sure to carefully read and follow the documentation to ensure your contribution can be reviewed and merged smoothly.

We truly appreciate every contribution that helps make the project more accessible to everyone. ❤️
